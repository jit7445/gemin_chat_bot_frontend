# Chat Application Frontend

A React-based frontend for a Gemini chatbot application.  
This application provides a web interface for interacting with an AI chatbot,
including text messaging and file uploads.
## Prerequisites

- Node.js (v18 or higher recommended)
- npm (or yarn)

---

##  Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/jit7445/gemin_chat_bot_frontend.git
cd gemin_chat_bot_frontend
npm install
npm run dev
